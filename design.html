Design Patterns Used in Implementation
The main design patters which are used in the implementation of the game of llife are follows:
  -- Model View Controller (MVC)
  -- Command

The use of these patterns are explained further in there respective sub-sections. The use of these patterns allows reusability of the code in different context without any major modifications.

In the game of life, buttons are used for each individual cell. This means that Cells are specilisation of buttons and have basic functionality which a button in java would provide but also has some other added features. The alllows higher maintainbilty since less things would be required to modify if in future more functions to the class Cell would be required. Also this means that  while adding cells to the grid is easy since it is a sub-class of JButton class in java.
